解决PHP-CGI爆增，造成VPS慢和无故死机问题

发表于 2012-04-23   |   分类于 学习笔记

很多用户会碰到 在任务管理器中 的PHP-CGI 爆多，造成VPS死机的问题

解决这个问题的办法是在web服务器配置中优化fastcgi配置文件参数。

在C:\WINDOWS\system32\inetsrv\fcgiext.ini 文件中可以设定php-cgi进程相关参数

打开这个文件， 在最下面的地方，找到
maxInstances=

把这个值改小
建议
512M 内存的改maxInstances=50
1G 内存的改maxInstances=80
2G 内存的改maxInstances=140

再修改
InstanceMaxRequests=

把这个值改小
建议
512M 内存的改InstanceMaxRequests=200
1G 内存的改InstanceMaxRequests=300
2G 内存的改InstanceMaxRequests=500

改完， 重启IIS，或重启VPS

phpstudy-css样式基础

发表于 2012-04-15   |   分类于 学习笔记 
**[从零开始学PHP](http://www.dudu.me "从零开始学PHP,phpstudy")**，是需要很大的毅力的，希望大家一起加油！遇到问题别退缩，以下是CSS的几种选择符.

1.类型(标签)选择符（控制整个页面下，凡是这个标签名字的都会生效）

所谓类型选择符，指以网页中已有的标签名作为名称的选择符。

如

body {}

h1 {}

p {}

2.群组（分组）选择符

除了可以对单个标签进行样式指定外，还可以对一组标签进行相同的样式定义。

如

h1, h2, h3, p {

font-size:12px;

font-family:arial;

}

使用逗号对选择符进行分隔。对页面中需要使用相同样式的地方，只需写一次样式。

3.包含选择符

当我们只打算对某个对象的子对象进行样式定义时，可以使用包含选择符。

如：

h1 span {

font-weight:bold;

}

4.id 及 class 选择符

id 及 class 均是css 提供由用户自定义标签名称的选择符，用户可以使用选择符id及class 来对页面中的xhtml标签进行自定义名称，从而达到扩展xhtml标签及组合html标签的目的。

如：id 选择符

<p id=“p1”> 这是一个段落 </p>

p1 {

font-size:12px;

font-weight:bold;

}

如：class 选择符

<p class=“p1”> 这是一个段落 </p>

.p1 {

font-size:12px;

font-weight:bold;

}

在网页中，每个id名称中只能使用一次，不得重复。

与id 不同，class 允许重复使用。比如页面中的多个元素，都可以使用同一个样式定义。

5.组合选择符

如：

h1 .p1 {} 表示h1标签下的所有class为p1的标签。

content h1 {} 表示id为content的标签下的所有h1标签。

6.伪类选择符

伪类可以看做是一种特殊的类选择符，是能被支持CSS的浏览器自动所识别的特殊选择符。它

的最大的用处就是可以对链接在不同状态下定义不同的样式效果。

 

a:link {color: #FF0000; text-decoration: none} / 未访问的链接 /

a:visited {color: #00FF00; text-decoration: none} / 已访问的链接 /

a:hover {color: #FF00FF; text-decoration: underline} / 鼠标在链接上 /

a:active {color: #0000FF; text-decoration: underline} / 激活链接 /

  • 一定要按照a:link， a:visited， a:hover， a:active的顺序书写

7.通配选择符 *

  • 号表示所有的对象

所谓通配选择符，就是指可以使用模糊指定的方式来对对象进行选择，指定样式。

如：

  • {

color:blue;

}

样式优先权

1.写法优先权

优先级依次是：（就近原则，越接近我们的标签的样式，优先级越高）

行内样式表 > 内部样式表 > 外部样式表

2.选择符优先权(控制范围越小，则优先权越大)

对于id 与 class 而言，id的定义优先于class 的定义。

具有class属性的比没有class属性的优先权要大。

id > class > 类型选择符

综合：行内>id>class>标签>内部>外部

样式继承

1.XHTML 中的子标签会继承部分父标签的样式。

如: body { color:#ff0000;}

那么页面中,body之下的所有标签及标签下的所有子标签的文本都将变成红色。

2. !important

在两行相同类型的css样式定义中，往往优先执行后面一个;

p {

font-size:14px;

font-size:20px;

}

 

但是可以通过!important 语法，提升某一句样式表的重要性，使及优先执先 !important标注的语句。

p {

font-size:14px;!important

font-size:20px;

}

css样式布局—探索篇

发表于 2012-04-11   |   分类于 学习笔记

1.什么是CSS?

CSS (Cascading Style Sheets) 层叠样式表

它能做些什么？

使用CSS样式可以有效地对页面进行布局。

使用CSS样式可以对页面字体、颜色、背景和其他效果实现精确描述，同时对它们的修改和控制变得更加快捷。

可以将站点上所有的网页风格都使用一个CSS文件进行控制，只要修改这个CSS文件中相应的代码，那么整个站点的所有页面都会随之发生变动。

CSS可以支持多种设备,比如手机,PDA,打印机,电视机,游戏机等

目的：将表现与结构分离。

2.CSS 语法结构

CSS 语法由三部分构成：选择符、属性和值

Selector {Property:Value;}

选择符 属性 值

例：

<style type=“text/css”>

 

body { background-color:#cccccc;}

</style>

3.将CSS应用到网页中

1.行间样式表

行间样式表是指将CSS样式编码写在XHTML 标签之中。

如：

<body style=“background-color:#ccccc;”>

<h1 style=“font-size:12px; color:#ff0000;”>这是一个标题</h1>

2. 内部样式表

内部样式表作为页面中的一个单独部分，由<style></style>标签定位在<head></head>之中。

如：

<head>

<style type=“text/css”>

body {

background-color:#cccccc;

}

</style>

</head>

3.外部样式表

外部样式表是CSS应用中最好的一种形式，它将CSS样式代码单独放在一个外部文件中，再由网页进行调用。

如：

style.css

body {

background-color:#cccccc;

}

 

<head>

<style type=“text/css”>

 

</style>

<link rel=“stylesheet”rev=“stylesheet”href=“style.css”/>

</head>

什么是PHP,php文件怎么打开,php文件用什么程序打开？

发表于 2012-04-10   |   分类于 学习笔记 
今天在QQ上一个网友问我**[PHP文件怎么打开](http:www.dudu.me "php文件怎么打开")**，怎么去浏览,我在这做个介绍吧.

对于许多没有任何基础的来说，的却是不清楚.关于PHP文件怎么打开，我们需要来清楚PHP几个问题：
一，php是什么：
PHP，是英文超级文本预处理语言Hypertext Preprocessor的缩写。
二，php是怎么工作的：
PHP 是一种运行在服务器端的一种语言程序，当用户浏览网页的时候，浏览器发送给服务器，然后服务器再回发给浏览器所能识别的HTML语言,最终展示在我们的眼前.
三，什么是php文件：
文件后缀为php的文件（*.php），比如：dudu.php
四，php文件打开的状态分为运行和修改状态（也就是编辑状态）：
运行：直接可以通过浏览器打开，服务器端需要有PHP的运行环境例如：LAMP、WAMP ,iis等才可以访问和运行。
编辑：记事本或者通过相关的编辑器如（Dreamweaver 、notepad++、editplus 等）打开即可编辑，对于前期不熟悉HTML的来说，还是选择记事本，有助于对指令的熟悉和了解。

CSS样式从零学-CSS样式表的三种引入方式

<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"&gt;

<html xmlns=”http://www.w3.org/1999/xhtml"&gt;

<head>

<meta http-equiv=”Content-Type” content=”text/html; charset=utf-8” />

<title>css样式表</title>

<!— css的页外引入，必须使用link标签来引用一个外部CSS文件，必须标明它的类型和关系，注意CSS文件的相对路径，通常我们会把外部引入的文件放到head标签里面，统一管理—>

<link href=”style/style.css” type=”text/css” rel=”stylesheet”/>

<!— css页内引入 注意样式必须写在style样式标签里面，不能写在style以外的任何地方，因为style标签以外的地方不属于HTML的语法规则，不支持HTML样式，只有style标签内部才支持style语法—>

<style type=”text/css”>

div{

color:#FFF;

background-color:#000;

 

}

/这个是CSS的注释，支持多行/

/CSS标签选择器，以元素的名字作为选择器的名字，它控制的范围是整个页面下凡是叫做该标签的标签，都会应用到该样式规则 /

p{

color:#36C;

background-color:#690;

}

</style>

</head>

<body>

<!—css的行内引入，注意样式必须写在该标签的style属性里面，并且一个标签只能够存在唯一的一个style属性 —>

<div style=”text-decoration:blink;bolor=@00FF00”> <!—blink代表的是文本闪烁，只有在非IE6下才有反应 —>
