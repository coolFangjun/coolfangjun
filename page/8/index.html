WordPress博客头像无法显示的解决办法

发表于 2012-03-15   |   分类于 网页教程 
WordPress是目前最流行的PHP博客系统，它免费开源，模板也比较多，到处都可以免费下载到。可如果你是中国大陆用户，那或许会碰到WordPress头像无法显示的问题，这主要是Gravatar头像被中国防火墙给档了，有什么办法可以解决啊？当然有哦。 一般来说，默认的WordPress头像的默认显示地址均为 0.gravatar.com 开头，如这个地址的头像是无法显示的： http://0.gravatar.com/avatar/8a050f3afce6fcf93367438777d1c17b?s=48&amp;d=&amp;r=G将0替换为www后发现问题解决，可以正常显示： http://www.gravatar.com/avatar/8a050f3afce6fcf93367438777d1c17b?s=48&amp;d=&amp;r=G解决办法： 话说在主题 functions.php 里添加如下代码即可解决。 function changeAvatarLinks($avatar) { return preg_replace('/\d+\.gravatar/i', 'www.gravatar', $avatar); } add_filter('get_avatar', 'changeAvatarLinks', 0, 1);
