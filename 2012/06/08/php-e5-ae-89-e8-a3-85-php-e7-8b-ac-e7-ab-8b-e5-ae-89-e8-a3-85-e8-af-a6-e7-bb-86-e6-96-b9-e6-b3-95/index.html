php安装-php独立安装详细方法

发表于 2012-06-08   |   分类于 学习笔记


一、准备好安装文件

1.apache_2.2.14-win32-x86-no_ssl.msi

2. mysql-5.5.13-win32.msi

3.php-5.3.1-Win32-VC6-x86.zip

4. phpMyAdmin-3.5.0-all-languages.7z

 

以上文件也可以到官方网站去下载：

php官网：http://www.php.net

mysql官网：http://www.mysql.com

apache官网：http://www.apache.org/

phpmyadmin官网 ：http://www.phpmyadmin.com/

 

二、关闭占用80端口的软件（如其他web服务器、迅雷、QQ等），不然可能会导致启动失败。

 

三、建立安装目录

 

1.为了统一管理，我们可以把这些软件全部安装在D盘的同一个文件夹下面，假设文件夹名称为amp

 

2.在amp文件夹里分别创建好php、apache、mysql文件夹,如图：


四、安装apache

 

1.双击apache安装文件,出现如图：



 

2.点击next,如图：



 

3.选择I accept，点击next,出现介绍页面，再点击next如图：



 

4.这一步使用默认选项就行了，默认作为一个服务从80端口启动。点击next



 

5.如果要自定义目录，这里我们可以选择下面那个选项（custom），点击next，出现选择组件和目录步骤，点击chang按钮选择我们准备好的安装目录（D:/amp/apache）



 

6.点击next,再点击install按钮完成安装。安装完成之后右下角任务栏会多一个任务图标,小三角形是绿色的代表apache启动成功了。



 

7.打开浏览器输入http://127.0.0.1 或者http://localhost 进行访问。如果看到it works就说明apache服务启动成功。

 

8.找到文件夹D:\amp\apache\htdocs 里面有一个index.html 用编辑器打开，就可以看到这个就是我们刚刚访问到的文件。试着改变里面的内容，再去访问http://127.0.0.1 或者http://localhost 看看有什么不同。

 

9:改变web服务器主目录：在D:/amp目录下新建一个www目录作为我们的web主目录。

打开apache的配置文件（D:/amp/apache/conf/httpd.conf）

177行：将 DocumentRoot “D:/amp/apache/htdocs” 改为

DocumentRoot “D:/amp/www

204行：将<Directory “D:/amp/apache/htdocs”>” 改为：

<Directory “D:/amp/www”>

重启apache，web服务器的主目录就从原来的D:/amp/apache/htdocs变成了D:/amp/www了,在D:/amp/www目录下新建一个index.html写点测试内容，再访问http://127.0.0.1。

 

10.到此为止apache安装好了，主目录也配置好了，接下来去安装PHP

 

 

五、安装PHP

 

1. php不需要像apache一样一步步安装。只需要把压缩包解压到我们的php目录（D:/amp/php）就行了。

2. 配置：

2.1 在php的安装目录找到php.ini-development、php.ini-production两个文件。随便复制一个文件并改名为php.ini（作为开发环境我们一般选php.ini-development）。

2.2 现在php.ini就是php的配置文件了。

2.3 php.ini809行附近，把; extension_dir = “ext”前面的分号去掉，并改为：extension_dir = “d:/amp/php/ext”

2.4 在apache配置文件中127行附近加入如下代码：

LoadModule php5_module D:/amp/php/php5apache2_2.dll

PHPIniDir “D:/amp/php”

AddType application/x-httpd-php .php .phtml

2.4 找到apache配置文件中的DirectoryIndex index.html 并改为

DirectoryIndex index.html index.php

2.5重启apache服务器。

2.6写一个PHP（index.php）文件放于d:/amp/www目录测试一下。

 

六、安装mysql

1.双击一直点击next,选择Typical


2.安装中…安装完成。

3.配置：



 

 

 

 


选择配置方式

Detailed Congiguration(手动精确配置)

Standard Configuration(标准配置)



选择Mysql服务器类型

Developer Machine(开发测试类，占用资源少）

Server Machine(服务器类型 占用资源较多）

Dedicated Mysql Server Machine(专门的数据库服务器，占有所有可用资源）

这里我们作为开发使用可以选择第一项。

选择Mysql数据库用途

Multifunctional Database(通用多功能型）

Transactional Database Only(专注于事务处理）

Non-Transactional Database Only(非事务处理型）

这里选择第一项。


选择好msyql的存放目录



Mysql同时连接的数目

Decision Support(20个左右）

Online Transaction Processing(500个左右）

Manual Setting(手动设置）

这里选择第一项。

是否启用TCP/IP连接，如果不启用，就只能在自己机器上访问mysql

mysql端口设定

选择语言编码

选择服务名，勾选环境变量

修改 root 用户（超级管理员）的密码 (开发用别设置太复杂了 123456就行了)

Enable root access from remote machines 是否允许root用户在其它远程的机器上登录

Create An Anonymous Account 是否创建匿名用户（匿名用户可以连接数据库，但不能操作数据库）

 

3. 在php.ini中加载mysql模块:

;extension=php_mysql.dll

; extension=php_pdo_mysql.dll

;extension=php_mbstring.dll

; extension=php_mysqli.dll

去掉这几个前面的分号

php安装的小笔记：

 

Apache的配置文件：Apache安装目录/conf/httpd.conf

安装完成之后浏览器输入：http://127.0.0.1 或者
http://localhost
更改web根目录：
更改Apache配置文件
DocumentRoot “D:/amp/www”
<Directory “D:/amp/www”>
改完之后保存并且重新启动Apache才能生效
PHP的配置文件：
PHP的安装目录/php.ini-development 或者 php.ini-
production

创建PHP配置文件：
php.ini-development => php.ini 或者
php.ini-production => php.ini
开启PHP扩展：
extension=php_gd2.dll
extension=php_mbstring.dll
extension=php_mysql.dll
extension=php_mysqli.dll
指定PHP扩展的目录：
extension_dir = “D:/amp/php/ext”
配置Apache支持PHP：
LoadModule php5_module D:/amp/php/php5apache2_2.dll
PHPIniDir “D:/amp/php”
AddType application/x-httpd-php .php .phtml
配置目录缺省首页：
DirectoryIndex index.html => DirectoryIndex index.html
index.php
上传临时目录设定：
upload_tmp_dir = “D:/amp/temp/upload”
设置可上传的文件大小：
upload_max_filesize = 200M
SESSION保存目录：
session.save_path = “D:/amp/temp/session”
测试mysql连接：

$conn = mysql_connect(‘127.0.0.1’, ‘root’, ‘123456’);
if($conn){
echo ‘连接成功’;
} else {
echo ‘连接失败’;
}

将以上代码保存为connect.php放于web根目录，然后去访问
